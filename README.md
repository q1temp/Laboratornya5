# Задание 1



## Задача 1



### Текст задачи



Найти наибольшую и наименьшую цифры в записи данного натурального числа. 



### Алгоритм решения

1) Входим в предикат go

2) Выводим на экран приглашение: "Enter a natural number (for example 123.): "  

3) Сбрасываем буфер вывода (flush\_output), чтобы сообщение сразу отобразилось

4) Считываем введенное пользователем число в переменную N через read(N)

5) Вызываем предикат min\_max\_digits(N, Min, Max) для поиска минимальной и максимальной цифры. В предикате проверяем N = 0, если да присваиваем min = 0, max = 0. Проверяем число однозначное 1-9, если да присваиваем min = n, max = n. если чсило многозначное, вычисляем последнюю цифру (остаток от деления на 10), отбрасываем последнюю цифру( целочисленное деление на 10), сравниваме текущую цифру D с min1 ( если D < Min1, То Min = D, иначе Min = Min1), сравниваем текущую цифру D с Max1( если D > Max1, то Max = D, иначе Max = Max1).

6) Проверяем результат: если предикат успешен (найдены Min и Max):выводим "minimum figure: " и значение Min, выводим "maximum figure: " и значение Max.

7) Завершаем выполнение предиката go

# Задание 5



## Задача 1



### Текст задачи



В одной школе уроки по истории, математике, биологии, географии,

английскому и французскому языку вели три учителя – Морозов, Васильев и

Токарев. Каждый из них преподавал два предмета.

– Географ и учитель французского языка – соседи по дому.

– Учитель биологии старше учителя математики. Морозов – самый молодой.

– В понедельник первый урок по расписанию у Токарева, у биолога и у учителя

французского языка.

– В воскресенье Морозов, математик и учитель английского язык были на

рыбалке.

Какие предметы преподает каждый учитель?





### Алгоритм решения



1) Входим в предикат go

2) Создаем список всех предметов

3) Выбираем предметы для Морозова, Васильева и назначаем предметы Токареву

4) Формируем список предметов для каждого учителя

5) Проверяем условия (логические ограничения)

6) Выводим результат

7) Завершаем выполнение предиката go
